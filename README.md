# homebrew-sandkasten

Homebrew tap for [sandkasten](https://github.com/DatanoiseTV/sandkasten) —
a fast, kernel-enforced application sandbox for macOS and Linux.

```sh
brew tap DatanoiseTV/sandkasten
brew install sandkasten
```

Release cadence: tagged releases on the sandkasten repo land here.
The `HEAD` install (`brew install --HEAD sandkasten`) tracks `main`.

See the sandkasten repo for documentation, profile schema, security
model, and disclaimer.
